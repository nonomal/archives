## 基础

插件

`vue-cli-plugin-docgen`

**install**

```bash
vue create my-project
vue add docgen
```

**use**

```bash
yarn docgen
```

`vue-presss`

```bash
yarn add -D vuepress # npm install -D vuepress
```
`vuepress`尽量用yarn安装

- [官方文档](https://www.vuepress.cn/guide/getting-started.html)
