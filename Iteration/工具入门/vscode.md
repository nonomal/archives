## 配置文件

根目录`.vscode/setting.json`

```json
{
    "i18n-ally.displayLanguage": "zh",
    "i18n-ally.localesPaths": ["./src/lang"],
    "i18n-ally.enabledParsers": [
      "js",
      "ts",
      "ini"
    ],
    "i18n-ally.keystyle": "nested",
    "path-autocomplete.pathMappings": {
        "@": "${folder}/src/",
      }
  }
```
